PAO System Reference Guide

About the PAO System

The Person-Action-Object (PAO) system is a powerful memory technique used by memory champions to memorize playing cards. Each card is associated with a unique person, action, and object. When memorizing cards in sequence, you create vivid scenes by combining these elements from three consecutive cards.

System Organization

The mappings are organized by suit, with each suit following a specific theme:

  • Hearts (♥): Story and fantasy characters
  • Diamonds (♦): Historical and professional figures
  • Clubs (♣): Greek mythology
  • Spades (♠): Modern icons and celebrities

Using the System

To use the system, take three cards in sequence. The first card gives you a person, the second an action, and the third an object. Combine these elements into a vivid, memorable scene. For example, if you draw AH-7D-QS, you would visualize "Shakespeare (AH) forging (7D) a microphone (QS)."
